Transfer files over to another user in real time.

Instructions

  • 1) The file is split into chunks which are then transferred via a WebRTC RTC Data Channel.
  • 2) Send and receive progress is monitored using the HTML 5 progress element.
  • 3) At the receiver end, file is reassembled using the Blob API and made available for download.
